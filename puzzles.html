Roulette Wheel Puzzle
Wrenwood Hotel · Parlor (Ground Floor)
Solution
RED — BLACK — RED — RED — BLACK
Spin the wheel to match the color sequence shown in the painting above the fireplace
Tip: The painting shows the sequence as dots. Left-to-right reading order. Red = ●, Black = ○. Reward: Parlor Key + Antique Coin #4.
  • 01 Enter the Parlor. Examine the painting above the fireplace.
  • 02 Note the dot sequence — 5 circles, each either filled (red) or empty (black).
  • 03 Interact with the roulette wheel. Spin to Red, then Black, Red, Red, Black.
  • 04 The locked drawer opens — collect the Parlor Key and Antique Coin.

Quartz Door Puzzle
Rhodes Hill · Central Hall — Quartz-sealed door
Symbol Sequence
☽   ☀   ★   ☽
Moon — Sun — Star — Moon
  • 01 Collect Moon Quartz from Chairman's Office safe puzzle (inspect the safe in the office).
  • 02 Go to the Quartz Door in Central Hall.
  • 03 Input the sequence: Moon → Sun → Star → Moon using the socket interface.
  • 04 Door opens. Behind it: East Wing Keycard + Upgrade ×1.
Don't miss: The Chairman's Office puzzle gives you Moon Quartz AND unlocks a separate reward. Solve the Chairman's puzzle before going to the Quartz Door.

RPD Power Grid Puzzle
Raccoon Police Department · Generator Room (B1)
Switch Order
C → A → D → B
Flip switches in this order to restore power without triggering the alarm
  • 01 Enter the Generator Room. Note the 4 breaker switches labeled A–D.
  • 02 The fuse diagram on the wall shows load order — read it top-right to bottom-left.
  • 03 Flip in order: C → A → D → B. Wrong order triggers alarm and spawns 2 enemies.
  • 04 Power restored: Elevator operational, Security Room door unlocked.

The Final Puzzle — ARK Core
ARK · Core Chamber · End of Game
Input Order (Grace's Terminal)
BLOOD · MEMORY · REQUIEM · REBIRTH
Enter the four words using the terminal keypad in this exact sequence
Input Order (Leon's Terminal)
TRUTH · SACRIFICE · ORDER · RELEASE
Both players must input simultaneously — if doing solo, Leon's terminal auto-locks after Grace completes hers
  • 01 The clues come from the Files collected throughout the game — Files #68 and #69 (Product Catalog) are the key ones.
  • 02 Grace's terminal: the words correspond to her personal memories as shown in the ARK's projection sequence.
  • 03 Leon's terminal: words come from the order of events in the founding document found in the Chairman's Office.
  • 04 Input both correctly to trigger the Release Elpis ending cutscene (the "true" ending).
Ending note: This final puzzle determines the ending branch. Inputting the wrong sequence on Leon's terminal triggers the alternate ending instead of Release Elpis. The codes above give you the true ending.
